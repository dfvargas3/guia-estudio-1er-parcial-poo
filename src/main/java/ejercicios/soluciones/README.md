## 💡 Consejos al revisar las soluciones

1. **No copies y pegues**: Entiende cada línea
2. **Pregúntate**: ¿Por qué se hace de esta manera?
3. **Compara enfoques**: Tu solución puede ser igualmente válida
4. **Busca patrones**: Validaciones similares, estructura similar
5. **Practica más**: Si no lo entendiste, intenta resolver otro ejercicio similar

## ✅ Checklist de comprensión

Después de revisar las soluciones, deberías poder:
- [ ] Explicar por qué los atributos son privados
- [ ] Justificar cada validación en setters y constructores

- [ ] Identificar los beneficios de usar `this()` entre constructores
